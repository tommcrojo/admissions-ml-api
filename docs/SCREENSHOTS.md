## Screenshot Guidelines

1. **Consistent Styling:** Use same theme/font for all screenshots
2. **Clear Labels:** Add captions explaining each screenshot
3. **Full Context:** Show column headers and relevant data
4. **High Resolution:** Use screenshots at 2x scale for clarity
